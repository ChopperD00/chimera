06 · STRAND
物語翻訳 · STORYBOARD TRANSLATION AND NOTATION FOR DESIGN

Draw what you see.
We build what you mean.

9-prefix unified tag language + 6-color ink system for physical storyboard notebooks. No competitor exists in this space.

>
Motion
fade, slide, scale, dither, squash, smear, arc, whip, shake
@
Timeline
seconds, frames, on-2s/3s, BPM sync, cuts, dissolves
#
Triggers
load, scroll, click, hover, viewport entry
$
Style
1-bit, CRT, film stock, anime, color grades, lighting
~
Reference
file refs, pose extraction, ~define, ~from, ~install
&
Character
identity, pose, expression, gaze, acting, silhouette
^
Camera
ECU–EWS, dolly, track, crane, orbit, lens, DoF
%
Routing
gen model target, tier selection, %local sovereign
!
Film tech
multiplane layers, cel assignments, key/breakdown, FX
6-COLOR INK SYSTEM
Pencil / Black
Composition — the actual drawing
Blue
Reference / onion skin — ghost of previous frame
→ ~ref:@prev-frame · ~extract:pose
Red
Motion / animation — arrows, speed lines, trajectories
→ >slide · >fade · &motion · ^pan
Green
Asset reference — character sheets, filenames, tokens
→ &char:name · &sheet:file.png · %pipeline
Orange
Interaction — click targets, hover zones, CTAs
→ #click · #hover · #scroll:0.3
Purple
Audio — music cues, SFX triggers, voiceover
→ %heygen · @audio:file.mp3
